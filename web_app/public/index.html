About the Project

This AI-Powered Smart Home Gesture Control System is a full-stack solution designed to provide contactless control over home environment. By bridging the gap between Computer Vision and IoT state management, the system offers an accessible alternative to tradition physical switches and voice commands.

HTML5/CSS3
JavaScript (ES6)
Node.js / Express
Python (Flask)
TensorFlow
MediaPipe
Supabase (PostgreSQL)

Dual-Stage AI Pipeline

Combines geometric detection (MediaPipe Hands) with custom CNN classification (MobileNetV2) for maximum accuracy.

Full-Stack Integration

Features a robust Node.js backend for authenticated sessions and a Python-powered inference engine.

Dynamic Performance

Processes real-time video frames with sub-500ms latency, providing instant visual feedback on appliance states.

Cloud Observability

Every gesture-triggered action is automatically persisted to Supabase, enabling a complete audit trail of device logs.
